# Componentes de terceiros

## TDLib

Este projeto inclui um binário Android `arm64-v8a` compilado do repositório oficial [tdlib/td](https://github.com/tdlib/td), commit `a17f87c4cff7b90b278d12b91ba0614383aaee82`.

A TDLib é distribuída sob a Boost Software License 1.0. O código-fonte e a licença integral estão disponíveis no repositório oficial.

O binário foi compilado localmente com a interface `JSONJava`, Android NDK `28.2.13676358` e OpenSSL `1.1.1w`, seguindo os scripts oficiais em `example/android`.
